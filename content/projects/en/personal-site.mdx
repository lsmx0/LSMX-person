# Personal Website

A fully simulated hand-drawn style personal website project.

## Tech Stack

- **Next.js 14** — App Router
- **Tailwind CSS** — Styling
- **Rough.js** — Hand-drawn SVG rendering
- **Framer Motion** — Animations
- **Supabase** — Guestbook database

## Features

- Dark/Light mode toggle
- Chinese/English bilingual support
- Hand-drawn line growth animation
- Tablet side-by-side book layout
